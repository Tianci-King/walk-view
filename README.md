# walk-view 精弘毅行 2021 报名系统前端

精弘网络在 2021 年决定尝试优化一下精弘毅行报名系统的性能，对原来的项目进行了重构



## 相关技术

**第三方库：**

- Vue

- Vue-router

- Naive UI

- axios

**前端构建工具：**

- yarn 包管理器
- Node.JS 运行环境
- Vite 构建工具



## 如何开始

**安装依赖：**

```bash
yarn
```

**本地运行测试**

```bash
yarn dev
```

**编译 realease 版本**

```bash
yarn build
```

**预览编译结果**

```bash
yarn serve
```
